## Tools and libraries

The notebook is built around:

- Python (via Jupyter Notebook).
- `pandas` for data loading, cleaning, transformation, and analysis.
- Optionally `numpy`, `matplotlib`, or `seaborn` for numerical operations and basic visualizations (depending on what you used in the assignment).

You can install these libraries using `pip` if they are not already available.

## How to run the notebook

1. Clone the repository:

   ```bash

   ```

2. Start Jupyter Notebook or JupyterLab:
